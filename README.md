# Iskakov Yeskendir
ToDo Web App with authentication in Django
Installation

Before starting to use the code you must install required packages and modules.

django (https://www.djangoproject.com/)
Usage
cd desktop
django-admin startproject todo_list
Examples
Firstly you should create superuser in the terminal, then login. To Do list web application with User Registration, Login, Search and full Create Read Update and DELETE functionality. You can create your own To Do List!—ç
